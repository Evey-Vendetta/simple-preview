# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this extension, please report it responsibly.

**Preferred:** Use [GitHub Security Advisories](https://github.com/Evey-Vendetta/simple-preview/security/advisories/new) to report privately without public disclosure.

**Alternative:** Open a GitHub Issue marked `[SECURITY]` for lower-severity issues that don't require private disclosure.

## Response Time

- **Acknowledgment:** within 48 hours
- **Status update:** within 7 days
- **Fix timeline:** depends on severity; critical issues are prioritised immediately

## Scope

This policy covers the extension source code in this repository. Vulnerabilities in VS Code itself, Node.js, or third-party packages should be reported to their respective maintainers.

## Out of Scope

- VS Code / VSCodium core bugs
- Issues requiring physical access to the machine
- Social engineering
